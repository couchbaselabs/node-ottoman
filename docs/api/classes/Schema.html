add​

▸ add(obj): Schema

Adds fields/schema type pairs to this schema.

Example

  const plane = new Schema({ name: String });
  const boeing = new Schema({ price: Number });
  boeing.add(plane);

  // You can add also add fields to this schema
  boeing.add({ status: Boolean });
